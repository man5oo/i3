# i3 dotfiles

Personal dotfiles for my laptop.

## Environment

- **OS:** Arch Linux
- **Window Manager:** i3
- **Bar:** i3blocks
- **Launcher:** rofi
- **Terminal:** alacritty
- **Notifications:** dunst
- **Compositor:** picom
- **File Manager:** ranger / pcmanfm
- **Theme system:** base16 / GTK
